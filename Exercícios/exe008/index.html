Principais formatações

Enfâse(Semântica = Significado e Forma)

Texto Marcado

Podemos usar a tag MARK para criar um texto marcado.

Também podemos usar CSS para mudar a cor do marcador.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.

Texto Sublinhado (não semântico)

Também existe a tag U para sublinhar — mas ela não tem valor semântico.

Texto Sobrescrito

Usamos a tag sup para escrever coisas como x20 + 3.

Texto Subscrito

Usamos a tag sub para escrever fórmulas como H2O.
