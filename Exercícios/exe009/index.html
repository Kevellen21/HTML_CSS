Lista de definição

HTML
HTML, ou Linguagem de Marcação de Hipertexto, é a linguagem fundamental para criar e estruturar páginas web. Ela define o conteúdo e a estrutura de um site, permitindo a organização de elementos como textos, imagens, links e outros conteúdos. O HTML não é uma linguagem de programação, mas sim uma linguagem de marcação que informa ao navegador como exibir o conteúdo da página.
CSS
CSS, ou Folhas de Estilo em Cascatas, é uma linguagem utilizada para descrever a apresentação de documentos HTML, XML e outros formatos XML. Em outras palavras, o CSS controla como os elementos de uma página web são exibidos na tela, como cores, fontes, layout e espaçamento.
JavaScript
JavaScript é uma linguagem de programação de alto nível, frequentemente usada para criar interatividade em páginas web, mas também pode ser utilizada no lado do servidor (backend) com o Node.js. Ela é a linguagem padrão que os navegadores interpretam e, junto com HTML e CSS, forma a base da experiência web moderna.
